## 这些功能真的很实用

### 1. @SaneLater - 稍后处理

这个文件夹专门收集那些"不紧急但可能有用"的邮件。比如newsletter、行业资讯、非紧急的内部通知等等。

我一般会在午饭后或者下班前集中处理这些邮件。一次性批量浏览，效率高得多。

### 2. @SaneBlackHole - 永久屏蔽

这是我最爱的功能。

把某个发件人的邮件拖进这个文件夹，以后他发的所有邮件都会直接进垃圾桶。再也不用手动点"取消订阅"，然后发现对方根本不理你的请求。

我已经往BlackHole里扔了300多个发件人。清爽。

### 3. SaneReminders - 邮件提醒

发出去的邮件石沉大海怎么办？用这个功能。

发邮件的时候，密送（BCC）一个特殊地址，比如 `2days@sanebox.com`，如果48小时内对方没回复，SaneBox会自动提醒你跟进。

再也不会因为忘记follow up而错失机会了。

### 4. Do Not Disturb - 专注模式

开会、写方案、需要深度工作的时候，开启DND模式。新邮件会暂时进入@SaneDoNotDisturb文件夹，等你空了再看。

不会被邮件打断思路，这个太重要了。

### 5. Deep Clean - 旧邮件清理

如果你的邮箱里躺着几千封旧邮件（就像我之前那样），Deep Clean能快速帮你分类整理，释放存储空间。

我用它清理了5000多封2年前的邮件，省出了2GB空间。

## 安全性怎么样？

这个问题很多人问。

SaneBox只分析邮件的**标题信息**（发件人、主题、时间戳），**从不读取邮件内容**。这也是为什么它能通过Google的OAuth API安全审核，并且拿到了SOC 2 Type II认证。

你的邮件内容永远不会离开你的邮件服务器，SaneBox只是在后台做分类整理工作。

隐私方面，不用担心。
